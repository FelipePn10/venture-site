Este projeto agora é uma aplicação Next.js (App Router) em TypeScript com Tailwind CSS. Para ver o site rodando, instale as dependências e suba o servidor de desenvolvimento na sua máquina.

# 1. instalar dependências
npm install

# 2. rodar em modo desenvolvimento
npm run dev

# 3. abrir http://localhost:3000
